# onlineShoping
Initialy Project setup using Gulp.

## Libraries used
* Angular.js : 1.4
* Ui-router : 0.2.15
* Bootstrap : 3.3.6

Other libraries are in the bower.json. 

## Build procedure
* Global libraries to have in the developer machine:
    - node.js
    - npm
    - gulp 
    - bower
* Run npm install
* Run bower install
